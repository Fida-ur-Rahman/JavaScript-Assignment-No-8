# JavaScript Loops: While & Do-While

## Overview
This repository contains study material and practice exercises on JavaScript loops, specifically `while` and `do-while`. It demonstrates how to perform repeated actions efficiently based on conditions.

## Concepts Covered
- `while` Loops  
- `do-while` Loops  
- Loop Control and Execution  
- Practical Examples  

## Usage
Run the JavaScript files in the browser console or using Node.js to practice and test loop examples.
